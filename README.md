# LLM-Static-Analysis

## Step 1: Build Your Virtual Environment
```
make build
```

## Step 2: Add OpenAI Credentials to `.env` file
```

ORG_ID=<ORG_ID>
OPENAI_API_KEY=<API_KEY>
```

## Step 3: Download and Filter the data
```
make data 
```

## Step 4: Run Test Experiment
```
make experiment
```
